Learning CSS

You have already learnt a little bit about CSS, however, here you will learn most of the key concepts to get all your websites looking slick. As mentioned earlier, CSS allows you to personalise your webpage, entirely!
Coding in CSS has slightly different syntax (code construction), and can be used in conjunction with HTML in 3 ways:

  1. Inline: Using a style attribute in the HTML tag itself
    <html>
      <head>...</head>
      <body>
        <p style="property:value;"></p>
      </body>
    </html>
  2. Internal: By using the style tag within the head
    <html>
  <head>
    <style>
	  element {
	    property: value;
	  }
	</style>
  </head>
      <body>...</body>
    </html>
  3. External: By linking to an external css file
    <!-- This is the HTML File -->
<html>
  <head>
    <link type="text/css" rel="stylesheet" href="YourStyleSheet.css">
	</style>
  </head>
      <body>...</body>
    </html>

/* This is the CSS file named YourStyleSheet.css (which is in the same folder)
element {
  property: value;
}

Container Tags: Divs and Spans

The <div> tag is a block element (an element that cannot be enclosed by another tag, aside from the body), used when you want to customise a certain section of your website. In order for your web browser to identify what content to change, the <div> encapsulates content and essentially (div)ides it into sections. When in your CSS file, for now, you can represent the <div> by simply typing in the tag: div. This is, sometimes, a useful feature, however, once you get to the 'Identifiers' lesson, you will find flaws with this method. After that you can then put in any attributes you like to make that div look the way you would like it to.

div {
  attribute: value;
}

You can also use any of the other ways of embedding/using CSS.
The <span> tag serves the same purpose as the <div> tag however it is an inline element (an element that sits happily within other elements). Using the span tag is identical to using the <div> tag. Keep experimenting using the editor on the right hand side.

Classes and IDs

We use the HTML tag attributes id and class refer specifically to a given element, or to a group of elements that may share the same styling. It is important to use classes and ids to show your browser where to find specific content when trying to make your website shine.

How do I know whether to use a class or an ID?

Classes are usually used when you are syling a group of elements. For example, let's say that you want all the paragraphs on your webpage, about dogs, to be green and underlined. Instead of individually going and giving each paragraph that styling scheme, you can simply add class="DOGS" to the paragraphs, and write your styling by refering to the class 'DOGS'. However, if you are styling one element exclusively, then it would be more appropriate to use an ID, as IDs are designed to be used for only one element. If you try using mulitple IDs with the same name, your website will probably glitch, and will not validate.

So, how do I use them?

Essentially, to add an HTML element to a CLASS, you must add the attribute 'class' to the tag, e.g. <p class="banana">BANANA!! :)<p>. To then refer to a class in the CSS document by putting a full stop, and then (with no space) typing in the class name.

.Banana {
  color: yellow;
}

To define an HTML element as an ID, you are required to add the 'id' attribute, as done in the class example, e.g. <p id="apple">APPLE!! :P<p>. Referring to an ID is similar to a class, however, instead of using a period or full stop, a hash or '#' is to be used.

.apple {
  color: red;
}

Common CSS Properties

There are a few extremely common CSS properties that you will almost always use when making your page attractive to the viewer. These elements are colour, border, font, margin, and background.

'Color'

The 'colour' propety is used to change the colour of text. You can reference colours by typing in their normal english name, e.g. 'yellow' or if you want to be more specific you can refernce a colour using its hex code or the standard rgb balance. REMEMBER that colour is spelt using the American spelling and therefore it is spelt ‘color’.

div {
  color: yellow;
  color: #FFFF00;
  color: rgb(255, 255, 0)
}

'Border-...'

The border represents the outside boundary of the website or selected element. You can alter and modify the border width, colour, and style. The 10 different border-styles are dotted, dashed, solid, double, groove, ridge, inset, outset, none, and hidden.

div {
  border-style: dashed;
  border-width: 3px;
  border-color: purple;
}

'Font-family'

Different fonts are used to make your text look interesting and change depending on the situation you are using them in. You can change font size, which is measured in ‘px’ (pixels). To change the font in your CSS file you would must use the 'font-family' property. Different fonts have different font families. These font families are used to help the browser find the correct font. If you are feeling fancy, you can use a special CSS function called @font-face. This is used when you want to use a custom font from the internet, or one that you have downloaded. Within @font-face, there are 2 properties you need to know, 'font-family' (this sets up the name you will use for the font), and 'src' (where you can retrieve the font file from to use it).

div {
  font-family: Helvetica, sans-serif;
}
		
@font-face {
  font-family: Custom;
  src: url('YOURFONT.ttf')
}
		
div.2 {
  font-family: Custom, sans-serif;
}

'margin' and 'align'

The margin and align properties in CSS will prove to be very useful in the positioning of elements on your webpage. For margin, there are 2 main value forms, percentages and pixels. There is another form of measurement, however, that can become complicated very quickly. If you are interested, search 'Viewport Measurement HTML'. Align; typically, the most used property using align is 'text-align'. Handy when trying to present information nicely.

div {
  margin-left: 47%;
  text-align: center;
}

'background-...'

The default colour for the background is white, however you can change the colour of the background to make your page more attractive. These 5 elements are essential if you want to make your website unique. You can even change your background to an image!

/*Whole page background*/
html {
  background-color: blue;
}

/* Div-specific background*/
div {
  background-image: url('YOURIMAGE.jpg');
}
